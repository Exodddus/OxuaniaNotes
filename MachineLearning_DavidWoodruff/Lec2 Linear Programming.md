## Min Cost Max Flow

![[Pasted image 20251018090348.png]]
在最大流问题的基础上，每一条边增加了一个权值作为cost，要使得通到终点的流量最大的同时cost最小。
- 方法一：先解最大流问题，然后将流量等于最大值作为一个新的条件限制，最小化成本函数。
- 方法二：增加一条从终点通向起点的边，容量为正无穷（极大值），成本为负无穷（极小值）。
	- 在实际操作中，这条边的成本可以取$-1-n*(max\{w(u,v)\})$


## Zero Sum Game

![[Pasted image 20251025010254.png]]

Gemini:
这两页PPT都在讲**零和博弈（Zero Sum Games）**，特别是如何为**行玩家（Row player）**计算**极小化极大（minimax）最优策略**，并将其表述为一个**线性规划（Linear Program）**问题。

**第一页PPT主要内容：**

- **主题：** 零和博弈和行玩家的支付矩阵（一个 $3 \times 3$ 矩阵的例子）。
    
- **目标：** 给定一个 $n$ 行 $n$ 列的零和博弈，计算行玩家的极小化极大最优策略。
    
- **变量：**
    
    - $p_1, p_2, \ldots, p_n$：行玩家选择每个行动的概率（混合策略）。
        
- **线性约束（部分）：**
    
    - $\sum_{i=1,\ldots,n} p_i = 1$（概率之和为1）。
        
    - $p_i \ge 0$（概率非负）。
        
- **目标函数（概念上）：** 旨在**最大化行玩家的最小预期支付**，这里的最小是针对所有可能的列玩家纯策略而言的。
    
- **求解技巧：** 引入一个新的“虚拟变量” $v$ 来代表这个“最小”值，从而将**最大化最小**的问题转化为线性规划。
    

**第二页PPT主要内容：**

- **主题：** 零和博弈（继续）。
    
- **符号定义：** $R_{ij}$ 代表行玩家选择行动 $i$，列玩家选择行动 $j$ 时的支付。
    
- **变量（正式化）：** $p_1, \ldots, p_n$（行玩家的概率）和 $v$（虚拟变量，代表行玩家保证能获得的最小收益）。
    
- **目标函数：** **最大化 $v$**（即最大化行玩家的最小保证收益）。
    
- **约束条件（正式化）：**
    
    - $p_i \ge 0$ 和 $\sum_{i} p_i = 1$（概率约束）。
        
    - **$\text{For all columns } j, \sum_{i} p_i R_{ij} \ge v$**（这是关键约束：无论列玩家选择哪个纯策略 $j$，行玩家的期望支付 $\sum_{i} p_i R_{ij}$ 都必须**大于或等于** $v$。这保证了 $v$ 确实是行玩家在最差情况下（即列玩家选择使其收益最小的行动时）的最小收益）。
        

**总结来说，这两页PPT提供了一个零和博弈中求行玩家最优混合策略的线性规划表述。**

这两页PPT继续讨论**线性规划 (Linear Programs, LP)**，重点是**标准形式 (Standard Form)**以及**可行域 (Feasible Region) 的几何性质**。

**第一页PPT主要内容：**

- **主题：** 线性规划的标准形式。
- **核心观点：** 任何线性规划都可以写成一个**标准形式**。
- **标准形式定义：**
    - **目标：** $\text{Max } c^T x$ (最大化一个线性目标函数)
    - **约束：** $\text{Subject to } A x \le b$ (线性不等式约束)
    - **非负性：** $x \ge 0$ (所有变量非负)
    - _注：这里使用了向量记号 $c^T x = \sum_{i=1,\ldots,d} c_i x_i$。_
        
- **如何转换到标准形式（将其他形式的问题转化为上述标准形式）：**
    - **处理等式约束：** 如何处理 $d^T x = e$ 形式的等式约束？（PPT中只提出了问题，没有给出完整的解答，但标准做法是将其拆成两个不等式：$d^T x \le e$ 和 $-d^T x \le -e$）。
    - **将最小化转换为最大化：** 如何将 $\text{Min } c^T x$ 转换为最大化？（通过最大化目标函数的负值：$\text{Max } (-c^T) x$）。
    - **处理无约束变量：** 如何处理可能为正或为负的**无约束变量** $x_i$？（代入 $x_i = y_i - z_i$，其中 $y_i \ge 0$ 且 $z_i \ge 0$。用两个非负变量的差来表示一个无约束变量）。
        
**第二页PPT主要内容：**

- **主题：** 关于线性规划的事实（几何解释）。
- **考虑的LP形式：** $\text{Max } c^T x$ 约束于 $A x \le b, x \ge 0$。
- **可行域 $A x \le b, x \ge 0$ 的几何形状：**
    - **半空间 (halfspace)：** 每一个约束条件（一行 $A x \le b$）在 $R^d$ 空间中定义了一个半空间，由一个超平面（hyperplane）分割而成。
    - **可行域：** 它是这些半空间和非负约束所定义的区域的**交集**。
    - 
    - **可行域的可能情况：**
        - **空集 (empty)：** 如果交集为空，则LP是**不可行的 (infeasible)**。
        - **无界 (unbounded)：** 如果交集向某个方向延伸至无穷大。
        - **有界 (bounded)：** 如果交集是一个有限的区域，我们称之为**可行域 (feasible region)**。
            
- **最大化目标函数的几何解释：**
    - **目标函数 $c^T x$** 可以看作是一个法向量为 $c$ 的超平面。
    - **求解过程：** 相当于**移动**这个法向量为 $c$ 的超平面，直到它**第一次与可行域相切 (tangent)**。相切点（或相切边、相切面）就是最优解。
